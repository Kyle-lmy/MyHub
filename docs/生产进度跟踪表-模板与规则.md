# 生产进度跟踪表 — 模板说明与填写规则

## 一、文件结构

建议在腾讯文档或 WPS 在线表格中建一个工作簿，包含 4 个工作表：

1. **订单总览** — 所有订单的当前状态汇总
2. **裁剪台账** — 裁剪每日进出流水
3. **印花台账** — 印花每日进出流水
4. **刺绣台账** — 刺绣每日进出流水

---

## 三、台账公式

结存列用公式，不需要手算。以裁剪台账为例，假设从第2行开始：

```
结存 = 上一天该款的结存 + 今天的进 - 今天的出
```

Excel 公式（E2 单元格）：
```
=IF(COUNTIFS($B$2:B2,B2)=1, D2-E2, SUMIFS($D$2:D2,$B$2:B2,B2)-SUMIFS($E$2:E2,$B$2:B2,B2))
```

或者更简单的方式：按款号分组，结存 = 该款所有进的总和 - 该款所有出的总和

```
=SUMIFS(D:D,B:B,B2)-SUMIFS(E:E,B:B,B2)
```

## 四、填写规则（打印出来贴在车间）

### 1. 谁填什么
- **裁剪部门**：每天下班前填裁剪台账，更新订单总览的"裁剪状态"
- **印花部门**：每天下班前填印花台账，更新订单总览的"印花状态"
- **刺绣部门**：每天下班前填刺绣台账，更新订单总览的"刺绣状态"
- **缝制部门**：每天更新订单总览的"缝制状态"
- **仓库**：入库后更新订单总览的"成品入库"

### 2. 什么时候填
- **每天下班前**，不管今天有没有生产都要填
  - 有生产：填进、出数量
  - 没生产：进填0，出填0，备注写原因（停机/等料/换款）

### 3. 进出的含义
- **裁剪**：进 = 投入面料对应的件数，出 = 裁剪完成的件数
- **印花**：进 = 从裁剪接收的件数，出 = 印花完成的件数
- **刺绣**：进 = 从上道接收的件数，出 = 刺绣完成的件数

### 4. 状态更新规则
- **未开始**：还没接到活
- **进行中**：开始投料/加工了
- **已完成**：订单数量全部做完，结存清零

## 五、日常管理动作

1. **每天早上 10 分钟对齐会**：打开订单总览，逐一过在产款的各环节状态
2. **每天下班前检查台账**：确认三个台账今天的记录都填了
3. **每周五做一次盘点**：台账结存 vs 实物结存，对不上的查原因
